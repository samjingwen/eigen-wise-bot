\begin{minipage}{0.65\linewidth}
Suppose $V$ is a complex vector space and $T \in \mathcal{L}(V)$. Which of the following linear operators necessarily has a square root?
\end{minipage}

\begin{enumerate}
  \item $T^*T + I$
  \item $T^*T$
  \item $TT^*$
  \item $T^*T - I$
\end{enumerate}
