How many possible distinct Jordan canonical forms are there for a $5 \times 5$ nilpotent matrix, up to rearrangement of the Jordan blocks?
\end{minipage}

  \item $5$
  \item $8$
  \item $7$
  \item $6$
\end{enumerate}
